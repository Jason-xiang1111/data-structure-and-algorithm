##  6-3 调整数组使奇数全部都位于偶数前面其他数字顺序不变
      关键点：数组通过函数的调用
              奇数偶数的分离
              string函数的使用
      关键代码：
   ```
      #include <string.h>
      void fun(char s[])
      {
          int length = strlen(s);   // 不计入结束'\0'字符
          int i = 0, j = 0, k = 0;
          char str1[21], str2[21];

          for(i; i<length; i++)
          {
              if((s[i]- '0') % 2 != 0)
              {
                  str1[j] = s[i];   //分离奇数
                  j++;
              }else{
                  str2[k] = s[i];   //分离偶数
                  k++;
              }
          }
          str2[k] = '\0';           //也要加入结束符

          strcpy(s, strcat(str1, str2)); // 字符串的连接、字符串的赋值

          printf("%s", s);

          return ;
      }
   ```
